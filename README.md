## GameBias

#### A social platform where users can track their emotional responses to playing video games.

### Features
* Upload a video
* View your emotional analytics

### Technologies used:
* Angular
* Node.js
* Express
* JavaScript
* Bootstrap
* Amazon S3
* Kairos API
* HTML5
* CSS
* Git
